# Uber Trip Performance Dashboard

## Project Overview
This dashboard analyzes Uber trip data to understand trip performance, fare trends, ride completion status, payment methods, and driver performance.

## Tool Used
- Tableau

## Key Metrics
- Total Trips
- Total Fare Amount
- Average Fare Amount
- Average Distance

## Dashboard Features
- Distance vs Fare Amount analysis
- Monthly trip trend visualization
- Completed vs Cancelled trip analysis
- Payment type distribution
- Driver performance comparison
- Interactive filters for:
  - Ride Status
  - Payment Type
  - Pickup Month

## Key Insights
- Most rides were successfully completed.
- Cash and Card were the most used payment methods.
- Higher trip distance generally resulted in higher fare amounts.
- Driver performance varied based on total fare generated.

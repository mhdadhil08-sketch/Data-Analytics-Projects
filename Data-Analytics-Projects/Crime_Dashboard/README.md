# Crime Analysis Dashboard

## Project Overview
This dashboard analyzes crime data across different cities to identify crime trends, category distribution, gender-wise analysis, and time-based crime patterns.

## Tool Used
- Power BI

## Key Metrics
- Total Crimes
- Average Age
- Total Cities
- Crime Categories

## Dashboard Features
- City-wise crime analysis
- Crime category distribution
- Crime trend over time
- Gender-wise crime comparison
- Time of day crime analysis
- Interactive filters for:
  - Crime Type
  - City
  - Gender

## Key Insights
- Fraud and Assault were among the most common crime categories.
- Crime trends fluctuated throughout the year.
- Crime occurrence varied across cities.
- Afternoon and Morning periods showed higher crime counts.

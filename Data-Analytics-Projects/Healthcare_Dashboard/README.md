# Healthcare Analysis Dashboard

## Project Overview
This dashboard analyzes healthcare data to monitor patient distribution, disease categories, hospital performance, admission trends, and treatment outcomes.

## Tool Used
- Microsoft Excel

## Key Metrics
- Total Patients
- Average Age
- Total Hospitals
- Disease Categories

## Dashboard Features
- Disease-wise patient analysis
- Hospital-wise patient comparison
- Outcome analysis
- Gender distribution analysis
- Admissions trend over time
- Interactive filters for:
  - Hospital
  - Disease
  - Admission Type
  - Gender

## Key Insights
- Diabetes and Asthma had higher patient counts.
- Some hospitals handled more patients than others.
- Patient admissions varied over different years.
- Recovery and treatment outcomes were effectively visualized.
